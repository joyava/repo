Install Ubuntu & Chrome

  • Download Rufus and the Ubuntu ISO on a Windows laptop.
  • Create a bootable USB drive.
  • Boot another computer from the USB (press F12).
  • Install Ubuntu.
  • Download Chrome (.deb) and install:
sudo apt update
sudo apt upgrade -y
cd ~/Downloads
sudo dpkg -i google-chrome.deb

Install Syncthing

Use Syncthing to sync files between computers.

sudo apt install syncthing
sudo systemctl enable syncthing@USER.service
sudo systemctl start syncthing@USER.service
  • Open: http://localhost:8384
  • Set username/password.
  • Add remote device using Device ID.

Install Ollama (Local AI)

sudo apt install curl
curl https://ollama.ai/install.sh | sh
ollama run phi3

AppImage on Ubuntu

sudo apt install libfuse2
chmod +x app.AppImage
./app.AppImage

Fix Overheating Laptop

Applying thermal paste reduced CPU temperature by ~10°C.

Eye Strain Tips

  • Lower brightness
  • Enable blue light filter
  • Use BenQ monitor settings properly (joystick control)

Slow Mouse Fix

If your mouse feels sluggish when connected to a dock, try using a USB extension cable to reduce interference.

Old Computer Use

Install Antix Linux for better performance on older hardware. Only .deb and .appimage apps work on Antix Flatpak and Snap apps don't work on Antix Steam games don't work on Antix You can make it work with Antix, but dependencies will slow Antix down.

DeepSeek 671B (Local Experiment)
