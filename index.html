Install Syncthing

Use Syncthing to sync files between computers.

sudo apt install syncthing
sudo systemctl enable syncthing@USER.service
sudo systemctl start syncthing@USER.service
  • Open: http://localhost:8384
  • Set username/password.
  • Add remote device using Device ID.
  • Example: on your browser type http://localhost:8384
  • Click setting, click GUI tab, Enter your username and password. Click save.
  • Repeat these steps on another ubuntu laptop2 (I have Ubuntu on a Raspberry Pi5)
  • Get ID from Ubuntu laptop1 and Enter that ID into Ubuntu laptop2
  • How do you do that? Click action, click show ID, copy to google keep this ID
  • On Ubuntu laptop2:
  • Copy ID code from google keep
  • click "add remote device" at the bottom of the page
  • You paste ID code into this pop up window. click save.
  • On Ubuntu laptop1:
  • click "add device" on pop up message
  • change device name to "laptop2". click save.
  • Now you sit back and watch the "remote device" box
  • When it shows "connected", you are done
  • type ls on your terminal
  • you should see a Sync folder along with Downloads and Pictures folder
  • The default Sync folder is at ~/Sync
  • The "default folder" entry on your browser shows /home/username/Sync
  • click "edit" button, it popup a windows with 3 tabs
  • click sharing tab, then tick the box
  • click file_version tab, choose "simple file version. click save.
  • On Ubuntu laptop2:
  • click share button

    DeepSeek 671B (Local Experiment)

    Running large models locally using swap is extremely slow and wears out SSDs quickly. Some people do this because they don't have a lot of RAM (more than 256G of RAM). Example: buy 2 TB SSD, install linux with 1T of swap. Then: ollama run Deepseek-R1:671b Then you wait for 12 hours for 1 query. From prompt to first output token, it takes 15 min. After 100-200 queries, your ssd is permanently damaged.

    My startup script for Antix Linux Distro

      
#!/bin/bash
# Menu script to pick utilities
clear
echo "Select a utility:"
options=("Update" "Browser" "Editor" "Quit")
select opt in "${options[@]}"
do
    case $opt in
        "Update")
            echo "Running update ..."
            sudo apt update && sudo apt upgrade -y
            ;;
        "Browser")
            roxterm --title="Browser" -e bash -c "google-chrome" &
            ;;
        "Editor")
            geany & 
            ;;
        "Quit")
            break
            ;;
    
        *) echo "$REPLY is invalid."; sleep 3;;
    esac

clear
echo "Select a number:"    
echo "1.Update, 2.Browser, 3.Editor, 4.Quit"
done
